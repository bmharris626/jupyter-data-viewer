# Mock Data Files for Testing

These files are used for testing the data viewer functionality:

- `test_data.csv` - Sample CSV data for testing file reading
- `test_data.tsv` - Sample TSV data for testing file reading  
- `test_data.json` - Sample JSON data for testing file reading
- `test_data.parquet` - Sample Parquet data for testing file reading

The data contains basic information including names, ages, and cities.
